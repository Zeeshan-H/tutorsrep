Hello Hamza

  • First goto https://tutorsrep.azurewebsites.net/api/account/register with a post request
    • You will notice that it does not support Get method, so open postman and hit same url. Note in header called body enter these credentials as json(application/json):
  • { "email": "any email",
  • "Names": "any first name",
  • "Gender": "Male or female",
  • "Domain": "either tutor or student",
  • "Dob": "Any Date of birth",
  • "Address": "Any address",
  • "password": "any password",
  • "confirmpassword": "same password"
    • }
  • You will be registered. Now go to https://tutorsrep.azurewebsites.net/signin with a post request
  • Under body header go to w-www-form-urlencoded, enter these values
  • username, value="the email you registered with"
  • password, value="the password you registered with"
  • grant_type, value="password"
  • You can also visit https://tutorsrep.azurewebsites.net/register.html
  • Just enter the credetials and register yourself
  • After that goto https://tutorsrep.azurewebsites.net/login.html
  • Now enter the username and password you registered with, you will be redirected to the page Data.html
  • Note if you dont type in the same password, you wont be redirected. So thats authorization
